Top Level Domain
Eine Top-Level-Domain (TLD) ist der letzte Teil einer Internetadresse nach dem Punkt, wie zum Beispiel ".com" oder ".de". TLDs dienen dazu, Websites eindeutig zu identifizieren und zu unterscheiden. Die TLDs werden von der IANA, einer Organisation zur Verwaltung von Internet-Ressourcen, verwaltet und an spezielle Registrierungsstellen weitergegeben, die die Verwaltung der TLDs übernehmen. Durch die Verwendung von TLDs können Benutzer auf einfache Weise auf Websites zugreifen und sie finden. Zum Beispiel kann man schnell erkennen, ob eine Website von einer Firma, einer Regierung oder einer Bildungseinrichtung betrieben wird, indem man sich die TLD ansieht.
